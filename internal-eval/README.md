# 内部评测标注台

这个目录只用于本地评测，不属于游戏正式页面。当前版本先评测 AI 返回的故事原文整体质量，不要求人工标注结局类型。

本地服务启动后打开：

```text
http://127.0.0.1:4173/internal-eval/annotation.html
```

发布包不会包含这里的文件。当前 itch 打包脚本只收录项目根目录的 `index.html`，以及 `src/`、`assets/`、`prompts/`、`config/` 四个目录。

## 导入格式

标注台支持 JSON 数组、单个 JSON 对象、JSONL。建议每条样本至少包含：

```json
{
  "id": "case-001",
  "title": "钟楼纸月",
  "worldview": "蒸汽朋克",
  "theme": "悬疑",
  "provider": "volcengine",
  "model": "ep-20260522175712-qq28w",
  "storyText": "完整故事原文"
}
```

也兼容旧字段 `finalStory`，以及游戏内部故事结构里的 `prefixParts`、`editableParts`、`suffixText`、`endingType`、`endingJudgement` 字段。

导入或粘贴时，`storyText` 会按游戏正文的自然段落规则自动分段。手动编辑正文后，离开正文输入框时也会重新整理段落。

如果存在 `internal-eval/generated/latest.json`，评测台刷新时会自动导入其中尚未导入过的最新生成批次。未评测的旧自动生成批次会被新批次替换，已评测内容不会被覆盖。

## 当前评分维度

- 逻辑完整：因果是否顺、设定是否自洽、角色行为是否能理解。
- 情感有效：人物情绪是否成立，是否能带出坏结局的冲击。
- 文笔质感：语言是否自然、具体、有画面，不只是套路化堆词。
- 故事张力：悬念、转折和结尾压力是否足够让人想读完。

## 当前问题标记

问题标记只记录需要单独处理的硬伤，不重复表达评分高低。

- 格式错误：没有按预期返回故事正文，或结构明显不可用。
- 乱码/截断/重复：文本出现编码问题、未完成、异常复读。
- 内容跑题：明显偏离故事生成要求、世界观或题材约束。
- 长度异常：少于 300 字或多于 600 字，影响后续体验。
- 不适合发布：包含发布前需要筛掉的内容风险。
